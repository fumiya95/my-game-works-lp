My Game Works

実際に遊んだゲームだけをまとめた、Webアプリ

1. 何を作っているか

  • 自分が3年間でプレイしたPCゲームのレビューサイト
  • 「おすすめ」と「おすすめしない」をはっきり分けて表示
  • カード型UIで、ゲームを探しやすくしたWebアプリ
ゲームレビュー Steam風UI 個人制作Webアプリ

2. 今できている機能

  • ログイン画面（簡単なユーザー名・パスワードでログイン）
  • 2025年ランキングTOP3の表示
  • カード形式のゲーム一覧表示（タイトル・ジャンル・おすすめ表示）
  • 「詳細」ボタンからレビュー本文と感想を表示
  • 6つのジャンルに分類して絞り込み表示
  • ゲームごとにスクリーンショット画像を表示できるようにしている

4. 作ろうと思ったきっかけ

  • 友達がゲーミングPCを買って「何のゲームを遊べばいいか分からない」と困っていた
  • 自分も、そもそも「面白いゲームに出会うこと」が難しいと感じていた
  • Steamレビューは情報量は多いが、最初の1本を直感的に選びにくい
  • そこで、自分が実際に遊んだゲームだけを集めた「入り口になるサイト」を作ろうと思った

5. 開発環境・使っている技術

  • HTML / CSS / JavaScript（プレーンな構成）
  • VS Code（ローカル実行）
  • 生成AI（ChatGPT）でUI案やコードのたたき台を相談しながら、自分で調整

6. これから追加したい機能

  • 「おすすめのみ」を切り替えて表示できるフィルター機能
  • 「参考になった」ボタン（いいねのような評価ボタン）
  • 背景色を選べるテーマ変更機能（ダーク／少し明るめなど）
  • ゲームの定価値段表示
  • 値段別に並び替え（安い順・高い順など）

7. この作品で伝えたいこと

  • 友達の「何を遊べばいいか分からない」という悩みからスタートした作品であること
  • 単なる日記ではなく、ゲーム探しの入口になる「自分専用のデータベース」を目指していること
  • 授業で学んだWeb技術と生成AIを組み合わせて、自分なりの形にできたこと
